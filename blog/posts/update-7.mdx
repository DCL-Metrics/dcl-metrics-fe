## Highlights

### Completed

#### Backend

- update `/scenes/top` endpoint to return top 50 daily scenes
- clean up some unused and out of date code
- update some api endpoints for greater flexibility
- release alpha version of client api
- work on pulling DAO governance data to enrich user information
- optimize sql queries
- fix broken worlds adapter

#### Frontend

- fix margin issues in rentals component
- update scenes list component to display top 50 scenes
- add historical data on scene dashboard pages
- work on dev blog
- display ~live concurrent users and active users from dcl data streams

## Blockers

No blockers!

## Next steps

After releasing the dev blog and concurrent users, we'll focus on shipping the
first major release for user dashboards (corollary to scene dashboards).

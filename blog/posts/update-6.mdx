## Highlights

### Completed

#### Backend

- add a few unit tests
- refactor some commonly used processes for better efficiency
- implement ability to get the previous' day data for any scene
- create pull requests for dao transparency data pulling (WIP)
- create pull request for returning data from top 50 scenes (in testing)

#### Frontend

- refactoring fetch component and cleaning up
- improve map functions including heatmap features and scene/parcel summary
- add component re: sales data
- add components re: rentals data
- fix various visual bugs related to scenes component
- add a search function to the map component to find any scene
- add a dev blog (WIP)
- updating scenes components to a sortable top 50 list view (in testing)

## Blockers

No blockers!

## Next Steps

our next planned release is updating the top scenes component into one sortable list.
that is currently in testing and we plan to deploy sometime in the week of the 19th
February. After that we will finish the dev blog and next we'll start working on
user dashboards + some tweaks and updates to sales / rentals components.

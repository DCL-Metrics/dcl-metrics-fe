# GET /users/:address

Fetch data about a user with a given address

## API ENDPOINT

### `https://api.dcl-metrics.com/users/:address`

### ex: `https://api.dcl-metrics.com/users/0xfe91C0c482E09600f2d1DBCA10FD705BC6de60bc`

#### SAMPLE DATA

```json
{
  "address":"0xfe91C0c482E09600f2d1DBCA10FD705BC6de60bc",
  "name":"Yemel#60bc",
  "avatar_url":"https://peer-ec1.decentraland.org/content/contents/bafkreidbx4b7wyp7qld47xr3iyiesdv6ptoqknmyrhcvqbsac4dygbmbiu",
  "first_seen":"2022-06-14",
  "last_seen":"2022-06-14",
  "guest":false,
  "verified":false,
  "dao_member":true
}
```

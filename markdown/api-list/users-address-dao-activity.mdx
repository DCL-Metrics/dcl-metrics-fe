# GET /users/:address/dao_activity

Fetch data about DAO activity of a user with a given address

## API ENDPOINT

### `https://api.dcl-metrics.com/users/:address/dao_activity`

### ex: `https://api.dcl-metrics.com/users/0xfe91C0c482E09600f2d1DBCA10FD705BC6de60bc/dao_activity`
